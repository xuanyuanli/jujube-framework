# Jujube Common Libs

Jujube框架的通用工具库集合，为Java开发提供各种实用工具和组件。

## 🎯 项目概述

该模块是Jujube框架的通用库集合，包含了多个独立的工具库和组件。每个子项目都专注于特定的功能领域，可以独立使用，也可以组合使用。

## 📦 包含的库

### [playwright-stealth-pool](./playwright-stealth-pool/README.md)
- **功能**: Playwright连接池管理和反检测功能
- **版本**: 3.1.1  
- **特性**:
  - 🔄 Playwright实例连接池管理
  - 🥷 内置反自动化检测功能
  - ⚙️ 灵活的配置选项
  - 🛡️ 健壮的错误处理和资源清理
  - 📊 连接池监控支持
